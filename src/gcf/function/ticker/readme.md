# Multiframework Data Processing Applications

This repository contains a set of applications demonstrating data processing capabilities using various frameworks and languages such as Spring Boot, simple Java, Micronaut, Python Flask, and Go. Each application includes the standard data processing elements but leverages the unique features of its respective framework.

## Applications Overview

### Spring Boot Application
A robust Java-based application utilizing Spring Boot for comprehensive microservices handling, integrated with Apache Beam for stream processing and Apache Spark for batch processing.

**Setup Instructions:**
- Navigate to the Spring Boot directory.
- Use Maven: `mvn spring-boot:run`

### Simple Java Application
A straightforward Java application that processes data in a traditional Java SE environment without additional frameworks.

**Setup Instructions:**
- Navigate to the Java directory.
- Compile: `javac App.java`
- Run: `java App`

### Micronaut Application
A lightweight, JVM-based application using the Micronaut framework designed for building modular microservices applications, integrated with Kafka for messaging.

**Setup Instructions:**
- Navigate to the Micronaut directory.
- Use Gradle: `./gradlew run`

### Python Flask Application
A Python application using Flask for creating a simple web service, with data processing powered by Apache Beam and Apache Spark.

**Setup Instructions:**
- Navigate to the Python directory.
- Install dependencies: `pip install -r requirements.txt`
- Run: `python app.py`

### Go Application
A Go application illustrating simple REST API creation using Gin and data messaging with Kafka.

**Setup Instructions:**
- Navigate to the Go directory.
- Run: `go run main.go`

# تطبيقات معالجة البيانات متعددة الأطر

هذا المستودع يحتوي على مجموعة من التطبيقات التي تظهر قدرات معالجة البيانات باستخدام أطر ولغات مختلفة مثل Spring Boot، وJava البسيطة، وMicronaut، وPython Flask، وGo. كل تطبيق يشتمل على عناصر معالجة البيانات الأساسية ولكن يستفيد من الميزات الفريدة للإطار الخاص به.

## نظرة عامة على التطبيقات

### تطبيق Spring Boot
تطبيق قائم على Java يستخدم Spring Boot للتعامل مع الخدمات الصغيرة بشكل شامل، متكامل مع Apache Beam لمعالجة البيانات المتدفقة وApache Spark لمعالجة الدُفعات.

**تعليمات الإعداد:**
- انتقل إلى دليل Spring Boot.
- استخدم Maven: `mvn spring-boot:run`

### تطبيق Java البسيط
تطبيق Java بسيط يعالج البيانات في بيئة Java SE التقليدية دون أطر إضافية.

**تعليمات الإعداد:**
- انتقل إلى دليل Java.
- اكتب: `javac App.java`
- شغّل: `java App`

### تطبيق Micronaut
تطبيق قائم على JVM باستخدام إطار Micronaut المصمم لبناء تطبيقات الخدمات الصغيرة المعيارية، متكامل مع Kafka للمراسلة.

**تعليمات الإعداد:**
- انتقل إلى دليل Micronaut.
- استخدم Gradle: `./gradlew run`

### تطبيق Python Flask
تطبيق Python يستخدم Flask لإنشاء خدمة ويب بسيطة، مع معالجة البيانات التي تتم بواسطة Apache Beam وApache Spark.

**تعليمات الإعداد:**
- انتقل إلى دليل Python.
- قم بتثبيت التبعيات: `pip install -r requirements.txt`
- شغّل: `python app.py`

### تطبيق Go
تطبيق Go يوضح إنشاء API REST بسيط باستخدام Gin والتراسل البياني مع Kafka.

**تعليمات الإعداد:**
- انتقل إلى دليل Go.
- شغّل: `go run main.go`
