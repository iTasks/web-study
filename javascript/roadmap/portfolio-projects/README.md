## Overview

These three projects cover the full breadth of the roadmap and are designed to demonstrate production-level engineering to EU employers.

| Project | Stack | Demonstrates |
|---------|-------|-------------|
| [Trading Dashboard](trading-dashboard.md) | React + Python + WebSocket | Real-time data, charting, distributed systems |
| [Service Marketplace](service-marketplace.md) | React Native + FastAPI | Mobile, payments, geolocation |
| [SaaS Admin Panel](saas-admin-panel.md) | React + Django + Next.js | RBAC, multi-tenancy, analytics |
